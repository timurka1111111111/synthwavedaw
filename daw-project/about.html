Что такое DAW
Digital Audio Workstation
Synthwave DAW — это полноценная цифровая звуковая рабочая станция, работающая прямо в браузере. Никаких установок, плагинов и дополнительного ПО — только открыл страницу и начал творить.

Проект вдохновлён профессиональными DAW: Ableton Live, FL Studio и Bitwig. Всё реализовано на чистом JavaScript с использованием Web Audio API.
Цель проекта
Курсовая работа
Проект создан как курсовая работа по дисциплине «Веб-разработка». Задача — показать возможности современного браузерного API и создать полнофункциональное интерактивное приложение без единой зависимости.

Основной акцент — на реальный звук: всё, что ты слышишь, генерируется прямо в браузере в реальном времени.

Как работает звук
Web Audio API под капотом
Весь звук строится через граф аудио-нод. Каждая нота — это OscillatorNode с огибающей (GainNode + ADSR). Ноды соединяются в цепочку:

OSC → Filter → ADSR Gain → FX Chain → Master Gain → Analyser → Destination Drum-звуки генерируются процедурно: кик — frequency sweep на OscillatorNode, снейр — белый шум через BufferSource + highpass фильтр, хэты — отфильтрованный шум с короткой огибающей.

Реверб реализован через ConvolverNode с процедурно сгенерированным импульсным откликом (Impulse Response). Delay — через DelayNode + обратная связь. Запись в WAV — через MediaRecorder API + декодирование и конвертация в PCM.
