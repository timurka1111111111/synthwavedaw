Частые вопросы
Вопрос
Почему нет звука?
Браузер блокирует Web Audio до первого взаимодействия пользователя. Нажми PLAY или любую кнопку — звук появится.
Вопрос
Как сохранить проект?
Нажми FILE в верхнем меню DAW → Save JSON или Save to Browser. JSON можно загрузить обратно через "Открыть проект" на главной странице.
Вопрос
Какой браузер нужен?
Любой современный: Chrome 90+, Firefox 88+, Edge 90+, Safari 14.1+. Web Audio API и MediaRecorder поддерживаются везде.
Вопрос
Можно ли использовать код?
Да, проект открытый. Смотри исходники на GitHub, бери, изучай, вдохновляйся. Ссылку на оригинал будем рады видеть.
